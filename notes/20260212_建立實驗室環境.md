## 階段一：建立實驗室環境

### 1. 實驗環境配置
- **虛擬機軟體:** Oracle VM VirtualBox
- **作業系統:** Ubuntu 24.04 LTS (Linux)

### 2. 學習重點與實作紀錄
今天我完成了 Linux 環境的初步架設，並學會了基本的軟體安裝流程。

#### 核心操作：
- **更新系統:** 使用 `sudo apt update && sudo apt upgrade` 確保系統安全性。
- **安裝工具:** 透過 `sudo apt install neofetch` 練習使用套件管理器。
- **排除故障:** - **問題:** 在輸入 `sudo` 密碼時發現畫面沒有反應。
  - **原因:** Linux 基於安全考量，輸入密碼時不會顯示任何字元。
  - **解決:** 直接「盲打」輸入密碼後按 Enter 即可成功。
