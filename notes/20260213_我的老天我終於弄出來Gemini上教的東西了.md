## 解決-1  
研究怎麼貼上（好像小學電腦課，但真的，我得研究這個

2. 按完後又去重新 sudo apt install build-essential dkms linux-headers-$(uname -r) 這坨  
3. 接著到處找這個檔案下載完跑去了哪裡  
4. 總之我是在虛擬機的光碟機裡頭直接兩下左鍵，然後重新開機  
5. 在登入畫面右下面的小齒輪改成 Ubuntu on Xorg 跟 VitrualBox 的 裝置->選單 勾了 雙向
6. 一通弄，反正現在是可以貼上了，不過 windows 的 VB虛擬機要用 control + shift + V
